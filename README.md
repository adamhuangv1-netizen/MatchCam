# Match Cam
Allows users to record and manage their tennis match recordings, by segmenting clips into a desired file size and zoom length. The app optimizes for battery saving while recording by introducing an auto or manual stealth feature, allowing the screen to effectively turn off while recording for long periods of time.
### How It's Made:
Tech used: Kotlin, XML, CameraX

The app is built around the Android CameraX library, which provides for low-level physical control over the users device.
